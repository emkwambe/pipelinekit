# PipelineKit CLI Commands

## Required MVP Commands

pipelinekit init

Creates a new PipelineKit project.

pipelinekit validate

Validates configuration, contracts, and blueprint structure.

pipelinekit run

Runs the configured pipeline.

pipelinekit doctor

Inspects pipeline state and explains health.

pipelinekit diagnose

Produces structured failure diagnosis.

## Rule

Every core capability must be accessible from the CLI.
