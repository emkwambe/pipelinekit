# PipelineKit Error Codes

## Format

PK-[AREA]-[NUMBER]

## Areas

CONFIG

CONTRACT

RUNTIME

ADAPTER

AI

STATE

## Examples

PK-CONFIG-001

Invalid pipelinekit.yaml.

PK-CONTRACT-001

Required column missing.

PK-RUNTIME-001

Pipeline execution failed.

PK-AI-001

AI provider unavailable.

PK-STATE-001

State database unavailable.
